# bachar-official.github.io
CV page with experience, education, hard skills, soft skills, courses, pet projects, hobbies, and language blocks. You can also generate a PDF with extended information.

## Why Flutter for this purpose?
Just curious how CanvasKit actually works.

## Compiling
- Ensure you have the Flutter SDK installed.
- Run the command: `flutter build web --no-wasm-dry-run --release`.
- Move the files from `/build/web` to the `/docs` folder.
- Commit & push.
- Wait until the build finishes.
